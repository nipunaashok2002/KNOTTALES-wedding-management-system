# Knottales
Excited to share KnotTales, a wedding planning & vendor discovery web platform I built largely from scratch as part of my academic and self-learning journey. 🔹 WHAT THE PLATFORM DOES  • Enables users to explore wedding vendors by category, budget, and location  • Supports booking enquiries with dynamic vendor selection  • Delivers a clean, responsive UI with a premium design aesthetic 🔹 WHAT I BUILT —( END TO END )  • Designed a normalized relational database with ER diagrams & foreign keys  • Implemented Django backend (models, views, URLs, templates)  • Used JavaScript (Fetch API) for dynamic, no-reload interactions  • Added client-side & server-side validation for data integrity 🔹 KEY CHALLENGES AND LEARNINGS  • Debugged dynamic dropdown issues using API-driven filtering  • Resolved real-world backend errors by aligning Django models with form data  • Gained hands-on insight into how frontend, backend, and database layers interact  • Built confidence in taking a product from idea → implementation → refinement
